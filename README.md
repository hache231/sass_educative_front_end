# Frontend - SaaS Gestion d'École

Application frontend Vue.js 3 pour la gestion d'établissements scolaires.

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

L'application sera accessible sur http://localhost:5173

## Build

```bash
npm run build
```

## Structure

- `src/api/` - Configuration Axios et services API
- `src/components/` - Composants réutilisables
- `src/layouts/` - Layouts (MainLayout, etc.)
- `src/router/` - Configuration Vue Router
- `src/stores/` - Stores Pinia
- `src/views/` - Pages/Vues
- `src/assets/` - Assets statiques (CSS, images)
